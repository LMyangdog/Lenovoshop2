lenovoshop 项目是一套电商系统，包括 lenovoshop 商城系统及 lenovoshop-admin 商城后台管理系统，基于 Spring Boot 2.X 及相关技术栈开发。 前台商城系统包含首页门户、商品分类、新品上线、首页轮播、商品推荐、商品搜索、商品展示、购物车、订单结算、订单流程、个人订单管理、会员中心、帮助中心等模块。 后台管理系统包含数据面板、轮播图管理、商品管理、订单管理、会员管理、分类管理、设置等模块。

  • lenovoshop 基于联想售卖网站进行开发，汲取了联想售卖官网的格式以及图片内容等；
  • lenovoshop 是一个 Spring Boot 大型项目，以newbee-mall为框架进行研发开源；
  • 可以把它作为 Spring Boot 技术栈的综合实践项目，lenovoshop 足够符合要求，且代码开源、功能完备、流程完整、页面交互美观；
  • 使用后可以简洁明了的看出各个商品的分类、价格、销量等；
